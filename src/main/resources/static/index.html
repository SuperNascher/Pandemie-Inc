Implementationen

  • Wir geben offiziell die "MedDeadlyFirstImplementation" ab.
  • BogoImplementation wählt immer zufällig eine Aktion von den verfügbaren Aktionen aus.
  • EndRoundImplementation wählt immer die Aktion "endRound" aus.
  • TriforceKIImplementation ist der "Implementationsselektor" von Kevin, welches probiert MedDeadlyFirstImplementation, MedFastFirstImplementation oder MedSlowFirstImplementation auszuwählen durch die Pathogene, welche am Start der 1, Runde auftauchen.
  • TriforceKIImplementation2 ist der "Implementationsselektor" von Jan, welches probiert MedDeadlyFirstImplementation, MedFastFirstImplementation oder MedSlowFirstImplementation auszuwählen durch die Pathogene, welche am Start der 1, Runde auftauchen.
  • Die Beschreibung der anderen Algorithmen befinden sich in der Dokumentation unter dem Punkt "2. Theoretische Lösung > 2.2 Lösungsansatz > 2.2.2 Strategie".
